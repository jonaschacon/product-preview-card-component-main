### What I learned

- Discovered a new property: `letter-spacing`.
- Using `rem` for font-size is better because I can set the font-size of my web browser, with pixels the font-size won't change.

```css
p {
  letter-spacing: 1rem;
}
```

```css
body {
    font-size: 1rem;
}
```
